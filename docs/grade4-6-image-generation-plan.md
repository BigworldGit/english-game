# 4-6 年级图片生成清单

更新时间：2026-03-29

这份清单用于安排 4-6 年级图片素材生成。已结合当前词表、现有图片目录、现有题池排除规则做过一次整理。

## 使用原则

- `建议生成`：优先安排，适合做单图猜词，且当前没有可直接用的图片
- `可选生成`：可以做，但必须给强场景，否则容易歧义
- `暂不建议生成`：不建议做成单图猜词，容易抽象、歧义，或和国籍/语言/短语混淆
- 文件命名：优先与单词完全一致的小写文件名
  - 例：`bedroom.png`
  - 多词词组如果后续仍要支持，建议和代码一起统一规则后再生成；当前不建议优先做短语图

## 二、可选生成，但必须给强场景

这些词不是完全不能画，但如果场景不够强，很容易误导用户。

- `quiet`
  - 建议场景：人物做“嘘”手势，旁边明确安静环境
- `active`
  - 建议场景：同一个孩子在跑、跳、打球等连续活动中

### 五年级

- `year`
  - 建议场景：完整日历年视图，而不是单月日历

### 六年级

- `learn`
  - 当前项目里已有图，但表达错误，已被移出图片题池

## 六、给素材生成同学的建议

### 房间/地点类

- 用明确室内布局，不要只画一个家具
- `bedroom` / `living room` / `kitchen` / `bathroom` 四者之间要有明显区别

### 动作类

- 人物动作必须单一明确
- 不要把多个动作拼在一张图里

### 活动/场所类

- `party`：明确生日帽、气球、蛋糕、多人庆祝
- `game`：明确游戏行为，不要画得像普通玩耍
- `theatre`：要有舞台、幕布、观众席等强提示，不要只画建筑外观

### 风格

- 延续当前 Minecraft / voxel / pixel art 风格
- 不要用大段英文文字充当语义主体
- 不要依赖标题词来让用户猜对
